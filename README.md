# Usage

You just need to run `./install.sh` and press Enter all the way! 

Needn't sudo.

The neovim config: [zrsharp/nvim](https://github.com/zrsharp/nvim)

# Support

- Arch Linux
- Debian
- Ubuntu

It mainly used in Arch Linux.
